# IKEA 3D Tour

A web-based 3D tour of IKEA showroom spaces. The project uses modern web 3D technologies (e.g. Three.js / WebGL) to present an interactive walkthrough.

## Features
- Interactive 3D showroom navigation
- Camera controls and hotspots
- Mobile and desktop-friendly

## Quick start

Prerequisites
- Node.js (16+ recommended)
- npm or yarn

Install
```bash
npm install
# or
yarn
```

Run (development)
```bash
npm run dev
# or
yarn dev
```

Build
```bash
npm run build
# or
yarn build
```

Serve production build locally
```bash
npm run serve
# or
yarn serve
```

Deploy
- This repository can be published via GitHub Pages or any static host. If you'd like, I can add a GitHub Actions workflow to build and deploy automatically.
